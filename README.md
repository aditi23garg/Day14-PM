# Advanced SQL Assignment

##  Overview
This assignment focuses on **advanced SQL concepts** used in real-world data engineering and analytics.  
It covers topics like **subqueries, window functions, CTEs, recursion, and query optimization**.

---

##  Files Included
- `pm_queries.sql` → All advanced SQL queries and solutions  
- `PM_SQL_Answers_v2.docx` → Detailed explanations and answers  

---

##  Concepts Covered

###  1. Window Functions
- Running total using `SUM() OVER`  
- Ranking using `ROW_NUMBER()`  
- Month-over-month growth using `LAG()`  
- Real-world use: revenue analysis and ranking customers  

---

###  2. Common Table Expressions (CTEs)
- Breaking complex queries into simple steps  
- Multi-step calculations  
- Improving readability and debugging  

---

###  3. Subqueries
- Correlated subqueries  
- Comparing values within groups  
- Example: finding second highest salary  

---

###  4. Recursive CTEs
- Generating number series (1 to 100)  
- Filling missing dates in time-series data  
- Useful in real-world reporting and analytics  

---

###  5. Time-Series Analysis
- Month-over-month growth calculation  
- Detecting missing data  
- Handling real-world incomplete datasets  

---

###  6. Query Optimization
- Understanding performance using `EXPLAIN`  
- Rewriting slow queries using window functions  
- Improving efficiency from O(n²) to O(n)  

---

###  7. Interview-Level Problems
- Consecutive month transactions  
- Ranking vs Dense Ranking concepts  
- Gap detection in datasets  
- Real-world SQL interview scenarios  

---

##  Key Learning Outcomes
- Strong understanding of advanced SQL concepts  
- Ability to write optimized and efficient queries  
- Knowledge of real-world data problems  
- Better preparation for SQL interviews
